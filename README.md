# ATTINY13 CLOCK

## Requiments
* [MicroCore](https://github.com/MCUdude/MicroCore)
* Arduino as ISP

## Components
- ATtiny13
- TM1637 
- DS1302
- Oscillator 32768Hz
- 1K and 10K resistors
- Push button
